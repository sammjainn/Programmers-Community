# Given an unsorted array of nonnegative integers, find if there is a continuous subarray with given sum.

# Constraints

- 1 < size Of Array <= 10^6
- 0 < Array Elements <= 10^9

# Input

- Array
- Its Size

# Output

- return "Yes" or "No"

```
Input   : { 1, 4, 20, 3, 10, 5 },sum = 33
Output  : yes 

```
