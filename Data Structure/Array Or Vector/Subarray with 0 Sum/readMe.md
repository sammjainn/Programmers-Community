# Given an array of positive and negative numbers, find if there is a subarray (of size at-least one) with 0 sum.

# Constraints

- 1 < size Of Array <= 10^6
- 0 < Array Elements <= 10^9

# Input

- Array
- Its Size

# Output

- return true or false

```
Input: {4, 2, -3, 1, 6}
Output: true 

Input: {4, 2, 0, 1, 6}
Output: true 

Input: {-3, 2, 3, 1, 6}
Output: false

```
